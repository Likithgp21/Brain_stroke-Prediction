**Brain Stroke Prediction**
This project is a Brain Stroke Prediction System built using Python, Flask, and Machine Learning.

Files Included:
1.	app.py: Flask web application for prediction
2.	HTML Files: Frontend files for input and result display
3.	Jupyter Notebook: For data preprocessing and model training
4.	requirements.txt: List of dependencies
5.	joblib models: Saved trained models for predictions
